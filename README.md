# Análises estatísticas e laboratoriais

No presente repositório, apresento diferentes códigos desenvolvidos no período de graduação. Em minha iniciação científica, em Químia Computacional, precisei desenvolver alguns scripts em python para conseguir análisar dados, extrair descritores moleculares, facilitar
contas, etc. Os códigos aqui desenvolvidos tiveram em grande parte ajuda de Inteligência Artifical para construção, visto que são códigos
grandes e muito difíceis de serem construídos. 

Não só isso, mas apresento também alguns códigos gerais de estatística e práticas laboratoriais.
